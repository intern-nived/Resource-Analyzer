# Resource-Analyzer

A simple prototype tool to help managers understand the resource requirements in a project.

## Deployment

To deploy this project,

Step 1: install the required packages
```bash
  pip install -r requirements.txt
```

Step 2: run the below code in terminal
```bash
  python analyzer.py
```
Side note: you can also edit and change the path in **run.bat** to make it run without going to the terminal
